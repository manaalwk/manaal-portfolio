# Manaal Portfolio (Uzair-style)

Minimal, recruiter-friendly portfolio built with **Next.js (App Router)** + **TypeScript** + **Tailwind**.

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Push this repo to GitHub
2. Import on Vercel
3. Deploy (no env vars needed)

## Customize
All your content lives here:
- `src/content/site.ts`

Resume PDF:
- `public/resume.pdf`
